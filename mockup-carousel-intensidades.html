DEMO INTERACTIVA

Tocá "LISTO" para completar la fase actual. Tocá "SIGUIENTE %" para avanzar a la siguiente intensidad con animación de carrusel.

Transición: La mini card upcoming se expande (scale .88 → 1, opacity .7 → 1, maxHeight 36px → full) mientras la card activa se colapsa (scale 1 → .88, opacity 1 → .55, maxHeight full → 36px). Ambas animaciones corren en paralelo con ease-out-expo (500ms).
Mapeo de botones (v1.7.3):
LISTO → skipPhase — completa fase actual (work→rest, rest→siguiente work, etc.)
⏩ → skipForward — salta a siguiente serie (o siguiente ejercicio si última serie)
SIGUIENTE → nextExercise — solo visible cuando NO quedan más % del ejercicio actual y existe un próximo ejercicio diferente
